++ 站规

注意：本沙盒站点是附属于 The Backrooms IF（下称“主站点”），用以撰写草稿的站点。本站与The Backrooms IF共用处分系统。
所有在主站点受到永久封禁的用户，都会在本站同步封禁；所有在本站受到的处分，都将同步至主站点。
[[/=]]
+++ 1.成员规范
任何人都可以在这里创建并编辑自己的页面，但你必须首先注册一个 Wikidot 账号并加入本站。而站点成员应遵循以下基本规范：

+++ 1.1.行为恰当
加入本站的成员应保持行为得体。必须尊重社区内的所有成员。提出反对意见或观点没问题，前提是必须保持礼貌。过于偏激、意图辱骂讽刺，或歧视他人者将会予以处分。同样，若在本站表现过于不成熟以至于干扰到他人正常使用站点，也会受到处分。

+++ 1.2.评分互动
本站允许进行评分相关的互动，但评分本身并不会有任何影响。对评分者而言，推荐考虑清楚后再给出自己的评分。对页面拥有者而言，不必过多在意相关的分数，不过可以将其作为一种可供参考的指标。若要进行互动，请确保您是 Backrooms IF 成员，并已加入本站。

+++ 1.3.关于对其他社群的攻击行为
本站为 The Backrooms IF 的附属站点，加入本站不等同于加入主站点。不得以本站或The Backrooms IF 的名义攻击另一个社群或站点。特别是在其他 Backrooms 相关网站内（如中文维基或 Fandom 维基，无论是否宣称自己代表本网站）对其网站用户或网站本身进行攻击、言语辱骂或是其他恶劣行为时，将会受到处分。

++ 2.内容规范
本站点的主要功能为撰写草稿，成员可在站点内创建属于自己的草稿页面，并在其中更新草稿内容、调试代码、接纳建议等。只要需要，你可以创建多个页面，但建议保持数目合理。页面及帖文内容应遵循以下规范：

+++ 2.1.具体内容
一般来说，草稿页面内容并没有任何必需规则。但若页面或帖文包含明确违反主站点内容政策的相关内容，例如引发强烈生理不适、涉及现实政治事件、对他人进行攻击等，站务将直接删除对应的违规页面或帖文，并给予发布者相应的处分。

+++ 2.2.命名和链接
草稿页面的URL和页面名都可按需指定，也不必需遵循主站点的相关命名规范。特别地，最好不要使用编号作为URL，例如“entity-if-1”或“level-if-100”等。此外，页面名和链接本身也同属页面内容的一部分，如包含不恰当内容依然会受到处分。

+++ 2.3.编辑
你可以随时修改属于自己的帖文和页面。但是，对本站内其他人发布的草稿页面，不论其处于何种分类，一般均不允许擅自编辑，除非得到了页面发布者的明确许可。

+++ 2.4.关于简体中文与繁體中文
根据个人习惯，在本站的页面和帖文都可使用简/繁两种形式，任何成员都不得因为语言差异而对相关语言使用群体提出质疑甚至歧视。本站及主站点都均非仅面向简中的维基。

++ 处分规则
[[/=]]
违反本站规则都将招致处分，处分内容由职员根据具体情况决定；所采取的行动会基于违反规则的具体情况和用户过去的行为。在本站所发生的所有处罚也都将被记录在职员站点的处分记录归档处，可供成员查询。站务一般会在讨论区/论坛帖子中发布处分帖文，也可能通过站内私信等方式告知违规用户受到处分。
处分共分为五档级别：{{提醒}}、{{警告}}、{{移除成员资格}}、{{封禁}}、{{永久封禁}}。此外，还有非处分、但同样带有警示意味的提醒。五档处分分别计数，且一经记录便不再移除。因此，我们诚挚建议你不要抱着“试试也无所谓”的心态违反规则。

+++ 提醒
提醒本身并非惩罚，它通常适用于成员行为轻微违规但影响小，站务认定不必处分的情况。此时站务会发送“职员帖：提醒”来提醒用户及时修正行为或作出相应措施。这是一种善意的提示：“好了，别再这样做了，下不为例。”累积提醒2次会改为1次警告处分。

+++ 警告
这通常是惩罚的最主要手段。受到警告将使你的警告计数 +1，受到严重警告则使你的警告计数 +2。如果警告计数达到 3 及以上，就有可能招致最高为移除成员资格的处分。达到 5 以上，就有可能招致最高封禁的处分。
以上行为都会让你受到警告，并且会以“警告”处分计入主站点处分系统：

+++ 移除成员资格
这是一个更严重的警告，但违规用户仍可以随时重新加入站点。这通常适用于以下情况：被警告后不听劝阻，忽略了所有警告并继续违规。
以下行为会招致移除本站成员资格的处分，并会以“警告”及以上的处分同步到主站点：

* **就简体与繁体问题对用户进行攻击。**任何就汉字繁简体进行攻击、嘲讽和无理由质疑的行为都将被直接移除本站的成员资格。
* **时政内容和政治攻击。**在网站内不得发布有关政治的言论，也不要试图在网站内就他人的政治、宗教等信仰或思想理念进行攻击，这是非常令人不适的行为。在发帖和页面中直接涉及敏感的时政事件、政治内容和政治主张的行为将会直接被移除本站成员资格。
* **骚扰或攻击（辱骂、嘲讽等）他人。**在任何情况下，本站都不允许有骚扰或言论上的人身攻击行为发生。如果你觉得某个用户正在骚扰/攻击你，请联系工作人员。在本站的任何帖文中都不应公然羞辱某草稿页面的作者，这会直接导致移除成员资格的处分，无论所不当评论的草稿页面本身质量如何。例如，阴阳怪气者也是处分的对象。这点请参见通告：关于嘲讽和辱骂低分文档及其作者的处分。
* **累计3次被警告。**累计3次违规并被警告将导致移除成员资格处理。

+++ 封禁
这一处分的范围从封禁1周到永久封禁。适用于严重违规，带有明显恶意或偏激行为的情况下作为最终措施。封禁是工作人员对于一些严重恶劣的行为或用户进行的最终警告，被封禁的用户在封禁期限内无法再次申请加入网站。
如果你有以下行为，你将受到封禁处罚，并会以“移除成员资格”及以上的处分同步到主站点：

* **多次被移除成员资格。**若因多次违规而被给予了移除成员资格处分，工作人员将考虑进行封禁处理。同样的，若你已多次受到短期封禁处罚，工作人员亦会考虑将你永久封禁。

* **歧视性发言和偏激行为。**发表种族主义等歧视性用语与发表宣扬支持纳粹/法西斯主义、恐怖主义/反人类或反社会言论是严重违反社群守则的行为，将会导致封禁一周的处罚。

* **人肉/开盒。**对网站用户进行人肉/开盒攻击是极为严重的骚扰行为，一旦确认，任何进行人肉的用户都将被永久封禁。同样地，帮助他人进行人肉、对人肉行为进行拱火或嘲讽受害者的用户将被封禁一个月。

* **账号分身。**任何试图以分身等方式绕过封禁的行为都会导致封禁延期。另外的，不应建立分身帐号重复加入本站。

* **冒充他人。**冒用他人身份是严重的侵权和欺骗行为。任何擅自使用他人的身份信息创建账号，或试图让别人相信自己账号的使用者为其他人的用户，一经发现，都将被永久封禁。
破坏页面/网站。诸如清空他人页面内容、添加攻击性代码等行为将被视为恶意破坏网站，这种行为将会导致封禁至少一个月的处罚。
