+++ 头版: 黑紫生产建设骑士团胜利解放镜厅

经过17.7小时的艰苦奋战，黑紫生产建设骑士团的战士们不负众望，终于在2022年1月15日下午彻底关闭了层级北面的阀门，取得了**开拓战争**的圆满胜利，彻底隔绝了层级中心地区可能受到的实体危害和环境危害。

根据**黑紫生产建设骑士团**代理团长**stormidry**透露，在此次行动中，**黑紫生产建设骑士团**共有42人阵亡，另有77人受伤，但黑紫生产建设骑士团的战士们利用热武器击杀实体至少7000名，取得了极其丰硕的成果。//我们在今天为牺牲者哀悼，他们的功绩将永世被镜厅的人民所铭记。//据透露，**开拓战争**的庆功宴将于1月18日于**安全区域**的中心举行。

+++ 边境旅宣布发起边境远征

早些时候，在层级北部的阀门完全闭合的前夕，在大团长**Sekistamd**的推动下，一支由50名精锐成员组成、且由大团长亲自率领的部队——即边境旅，已携100升**抗凝剂**越过北部的阀门，宣布发起**边境远征**。此举可能将开拓**安全区域**的界限，亦或只是**集体性的自杀罢了**。

关于**黑紫生产建设骑士团**的内部事务已经提前托付给副团长——即现代理团长**Stormidry**管理。至于事态将如何发展，目前还尚未存在定论。请等待我们进一步观察的结果。

+++ “因特尔温集市”工作委员会揭幕仪式

2022年1月16日，23位**“因特尔温集市”**的商户宣布 “因特尔温集市”工作委员会正式于层级南部主干道中成立，并推选**黑紫生产建设骑士团**代理团长**Stormidry**当选第一任工作委员会主席。

本次活动终结了**“因特尔温集市”**长久以来没有官方管制的历史，具有历史性的意义，再次确立了本层级“自由贸易、有序贸易”的根本制度。

**Level-IF-177**是后室IF层群的第177层。
+ 描述
**Level-IF-177**表现为一座由一系列大量异常巨大的管道组成的排水系统，其管径通常为5～25m不等，可供人通过甚至建立据点等。管道内壁大体为黑色，通体光滑，部分管道底部存在积水或冰。据研究表明，其材质类似于PVC管道，但尽管如此，将其破坏似乎是极其难以做到的。故我们尚不清楚管道之外的情况，但推测为虚空。

层级内不存在自然光源，其室温在-2～7摄氏度之间波动，空气拥有较大湿度，这可能将导致当室温低于零摄氏度时，空中将存在大量漂浮样的冰晶，这些冰晶由于未知原因异常锋利，此时请尽量佩戴口罩以避免吸入冰晶从而导致食道或呼吸道被割伤。该层级内存在稳定的4GWI-FI信号。

有时，流浪者会发现部分管道内壁上附着着一层幽蓝色粘液，请及时向**黑紫生产建设骑士团**上报，这是十分珍贵的资源，我们暂且命名为**抗凝液**。据研究表明，该种物质似乎可以使物体避免任何形式的凝固；当流浪者服用时，将保证其可在不低于零下217摄氏度的条件下生存至少72小时并使其体温将保持在35摄氏度以上。目前该种资源正在被**黑紫生产建设骑士团**大量开采并出口，这将保证处于其他层级的流浪者在极端寒冷环境下的生命安全。

+++ "阀门"
在本层级内，每隔一段距离都将存在一个类似于前厅中卷帘门的结构，但其同管道本身一样，被严重放大化。值得注意的是，阀门可以从内部被关上，但需要相当长的时间，这一过程可能将持续24～36小时。一旦阀门闭合，任何生物都难以通行，这一范畴包括层级内的实体。

在**我们划定的安全区域**内，所有的**阀门**均处于开启状态下，以保证层级内的正常交通。在**安全区域**的边缘，存在数个闭合的、被**抗凝液**涂满的**阀门**已确保任何实体或外部环境都难以影响到层级中央的情况。**请不要**试图改变任何**阀门**的开闭状态，否则这**极有可能**将对层级内的正常活动造成**极其严重**的破坏。

+++ "安全区"
无论从哪个入口进入，流浪者必然将切入该区域的几何中心，这也是我们能够保证隔绝实体与外部环境的根本依靠。该区域的中心地区表现为一组巨大的管道交点，其长宽高约为150*150*25，向四个方向分别延伸出一条管径为20m的管道并延伸500m，随即可见到上文提到的特殊阀门。此即**安全区域**的边缘。

在**安全区域**的范围内，可能会存在分岔路口，但事实上它们全部通向死胡同。因此，请放心，在**安全区域**内，**不可能**存在**任何实体**。

+ 边缘区域

**边缘区域**是**安全区域**的延伸。

当流浪者离开**安全区域**的最后一道阀门，他将会来到**边缘区域**。其外观与**安全区域**在事实上完全相同——不如说其本就是同一个区域，**开拓战争**的战线决定了两个区域的划分。本区域由于无**阀门**的阻隔，直接受到寒潮的侵袭，其温度多在-27～-55摄氏度之间，最低观测到-115摄氏度的低温。

在**边缘区域**的**主干道**上，将大量刷新下文所述的实体**霜行者**。这一切使得在该区域的生存变得极为困难，哪怕是最最精锐的流浪者也难以在该区域单独生存5分钟以上。

目前，该区域与**安全区域**之间隔离的每个阀门都至少受到3名**黑紫生产建设骑士团**成员的看守，以避免任何未经许可的流浪者进入该区域。

+ 物资与实体

在本层级中，仅存在**抗凝剂**一种物资，但得益于其稀缺性与实用性，本层级得以凭借该物资的大量出口吸引大波流浪者前来进行交易，连带产生了大量定居点，在相互促进的作用下，交易的范畴不再止于**抗凝剂**，现在本层级已经成为**IF层群**中的大型贸易港。

在本层级的**边缘区域**，一种特有的、致命的实体广泛存在，我们称其为-**"霜行者"**。

+++ "霜行者"

迄今为止，我们为了对抗霜行者已经遭到了极大的破坏。它们从四条主干道中涌来，四肢由冰组成，其形体大抵接近于人类，但移动方式极其诡异，类似于蠕动与行走的结合，但却惊人的能爆发出接近23km/h的速度。该实体似乎不存在意识。其力量远超于常人，且数量异常巨大。在**开拓战争**期间，在一个标准时内最多有记载刷新了700只**霜行者**。能够杀死它们的，只有**热武器**。该实体极端畏惧光亮，因此现在**安全区域**已经铺设了大量led灯光以预防未来可能在失去阀门保护后该实体的侵袭。

**唯有光与热可以与之抗衡。**

+ 据点、社区和前哨

++++ "黑紫生产建设骑士团"总部

该组织由300名左右流浪者组成，开放并积极组织贸易，其对于层级内的探索与基础设施建设有极大贡献并曾付出过极大代价。目前，该组织已经具有相当武装力量，主要维护层级内的治安、照明等，并正试图扩展**"安全区域"**的范围。该组织在本层级内经营有**抗凝液收集基地**，目前其**抗凝液**储量已达到5,000升左右，并积极向外出口。**兑换15毫升抗凝剂仅需一瓶杏仁水**。除此之外，该组织正积极支援其他层级——特别是**环境危害**层级的探索。

++++ “边境旅”

如上文所述，该组织是一支由**黑紫生产建设骑士团**大团长亲自率领的部队，大约有50人左右，在2022年1月15日越过**最后一道阀门**后彻底失踪，至今未重新建立联系。推测已经全军覆没。

++++ “因特尔温集市”

该组织由70名以上来自后室各层级的商人组成。他们活跃于各个主干道上，积极与流浪者交换各种物资，是该层级繁荣的根本，也是层级内获取补给的唯一途径。

+ 入口和出口

+++ 入口

1.在绝大多数层级中，你都可以找到一个**突兀的黑色排水管道**，顺着其寻找，可以找到其排水管口，接触它可来到本层级。
2.在本层级的底部，偶尔可以看到一节**黑色的PVC管道**，你可以将其带走，在其他层级切入其中可以回到本层级。不用担心，它会在本层级**定期刷新**。

+++ 出口

1.若你在该层级中所处的位置足够明亮，切出管道壁可以来到任意一个**环境危害**的层级。
