+ **描述：**

**层级探索器**为一套连接流浪者动作与意识的VR设备，包括一个头盔、两个手传感器、脚传感器、上身传感器和下身传感器，设备内部包含触觉传感器。此物品连接着一个仿真人体，仿真人体通常被认为是**层级探索器**的一部分，当流浪者使用**层级探索器**时，仿真人体将会作出与流浪者一样的动作。

**层级探索器**的头盔有黑白两色，表面较为光滑，内部为黑色，质感较软，装有触觉传感器，头盔长宽可调节，眼部位置为一个小型显示屏，可以通过手传感器挥动打开菜单来调节色彩模式，也可以通过使用手传感器做出打响指动作开启沉浸式模式。手传感器和脚传感器可以随意取戴，包括一块大传感器和绑定区域，绑定区域为柔韧材料制作，在穿戴时可以调成曲度。上身传感器和下身传感器同样为大传感器和绑定区域组成，穿戴时绑定区域要围住腰或胸部，可以调整松紧度。

仿真人体为肉色塑料制成，外观类似于没有任何特征的人体形状，类似于人体模型，分为男女两种，但不论是男女仿真人体都可以穿戴，仿真人体的组成部分未知，触感类似于人类皮肤，在上面标有黑点，黑点为流浪者穿戴传感器对应在仿真人体的位置。仿真人体可以进行类似于人类的任何动作且可以切出，当仿真人体受到痛觉和触觉时同样会在流浪者身上产生轻微的感觉，可以调节触觉的灵敏度，当仿真体受到严重损坏时，流浪者将会短暂眩晕，当仿真人体进入可能造成流浪者生命危害的层级或是流浪者心率过快或过慢时，设备将会提醒流浪者卸下**层级探索器**
+ **用途：**
**层级探索器**顾名思义，可以探索任何未知层级，在探索时也无需流浪者发生意外情况或者受到巨大损伤，且此物品大部分区域都有分布，但生成的极少，部分为人为制造。**层级探索器**可以与其他电子设备进行连接，最广为流传的用法为与[[[https://if-backrooms.wikidot.com/object-if-3|万维聊天室维基]]]进行的连接，当与此设备连接时，流浪者将可以使用手传感器进行打字和查看聊天室，且可以开启一个**层级探索器**的独占的特殊模式：直播层级。当开启直播层级后，其他流浪者进入此主题帖将可以实时观看此流浪者的探索过程。
