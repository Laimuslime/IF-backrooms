++ 竞赛规则
# 允许使用已有的灵感或草稿参赛，只要它们从未在网站上发布过。
# 每个人可以发布任意多篇参赛作品，但仅有一篇其得分最高的参赛作品参与排名；允许发布合著作品，但若获得奖品，奖品只能给予所有合著者的一名，主办方不负责分配。
# 竞赛期间所有作品将遵循正常的低分删除规则。除非页面崩溃，不允许自删竞赛投稿，否则所有投稿均取消参赛资格（合著除外）；不允许在进入截稿投票之后对自己的任一投稿作出实质性修改。
# 严禁任何不正当竞争行为，站务有权对任何已确认的不正当行为进行适当的处置，包括对不正当竞争者采取取消比赛资格、警告、取消成员资格等措施。
# 不允许通过 QQ 私聊、在群聊中发布文章链接、Wikidot 私信等方式推荐自己或他人的已发布的竞赛文，直至本次竞赛的投票结束。违者将由站务处理。
# 除作者页的 listpages 模块之外，不得在竞赛结束之前将你的投稿链接发布在其他 Wiki 页面上，违者认定为不正当竞争。
# 完成你的投稿作品后，待进入投稿阶段，则如通常页面一般发布你的参赛作品（此次竞赛无需提前报名），并加上 {{竞赛标签}} 标签。
# 参加竞赛的文章在参赛期间遵循 **CC BY-NC-ND** 协议，直至竞赛结束。请来自站外网站的制作者自觉遵守规则，在竞赛期间不宣传竞赛作品。
# 提前了解竞赛主题的成员会按照规定被扣分；在本次竞赛中，这条规则适用于[[*user 用户名]]、[[*user 用户名]]、[[*user 用户名]]，若他们参赛，参赛作品只会按照原有分数的 93% 计分。
