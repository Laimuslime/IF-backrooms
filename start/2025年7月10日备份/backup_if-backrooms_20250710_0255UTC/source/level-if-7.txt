Level IF-7是IF层群的第7层。


+ 描述

   自首次勘探起，**Level IF-7**完全处于大雾之中，在层级的不同区域中，气温，能见度、层级效应与WiFi强度等特质各不相同。唯一可以确定的是，该层级的天空中存在稳定的未知光源，其光照强度类似于前厅中二月左右午后的太阳。该层级不存在昼夜更替。

   截止目前，没有任何报告表明在该层级内有机会遇到其他流浪者，因此我们判断该层级大概率存在孤立效应。

   该层级内存在广泛的雨夹雪天气。尽管层级内环境温度普遍低于零摄氏度，但雪花依然难以在地面上留存。

   值得注意的是，在该层级已勘探的区域中，未发现任何物资与补给，故不推荐任何流浪者在该层级长时间停留。

   由于大雾的存在，我们难以对该层级进行大规模探索，仅能确定存在**回望步道**与**大厅**与**绿化林**三个区域的存在。

++ 回望步道

   根据大多数流浪者的报告，他们将在进入层级后出现于该区域。该区域表现为一条漫长的石砖路，部分路面的石砖已经存在裂缝，足以体现出其年代感。石砖路的两侧存在铁质垃圾桶，但其不存在任何使用痕迹，其中也不任何垃圾或可利用物资。

   大雾下的能见度约5m，本区域入口附近的环境温度约为-4℃，其长度难以估计，宽约12m，石板路两侧似乎是一片茂密的针叶林。

   任何试图离开石砖路的行为都是极其难以实施的，当流浪者试图离开石砖路时，似乎有一股无形的力量迫使流浪者回到石板路上。

   流浪者在石板路上向前行走时将感到强烈的安定感与怀旧感，反之将感到强烈的恐惧，故流浪者难以在此处折返，只能向前行走。由于层级特殊效应，我们难以预知折返的后果。随着流浪者的深入，环境温度将进一步降低。

   进入该区域的流浪者几乎全部报告，自己在进入该区域一段时间后，自己听到一段未知来源的关于天气预报的收音机广播，内容基本如下:

“各位观众朋友大家好，感谢各位对本台的支持，下面开始进行今日天气预报：今日［不存在统一报告］地区最高气温为-4℃，最低气温为［报告为杂音］℃，预计于14时左右开始降雨夹雪。预计降雪将持续到［报告为杂音］
[广播逐渐变为杂音并消失]”

   该现象似乎会使流浪者产生某种熟悉感，且其中关于广播中“地区”一处的描述往往与流浪者在前厅中居住的地点相重合。但根据部分流浪者随身携带的分贝仪的记录，并不存在所谓的广播。

++ 大厅

   当流浪者在**回望步道**中经过数分钟至数小时的行进，流浪者将离开**回望步道**，进入**大厅**区域。

   该区域表现为一座普通的别墅小区，能见度约为8m，进入此区域时，温度约为-8℃，温度将随时间下降，最低观测到-57℃的温度，不建议流浪者在此长时间停留。

   本区域中的流浪者将感受到强烈的温馨感和怀旧感，这种感觉不会因任何外因而消失，流浪者可能因此忽略区域内的寒冷，从而引起冻伤。

   流浪者进入该区域后，经过一段时间的行进将进入其腹地，其表现为一座绿草覆盖的小山，其能见度将进一步提高至约20m。小山的顶部有一座亭子，其左侧种有一颗类似前厅中樱花树的植物，目前所有的报告都说明其正处于盛开的状态。

   在大厅区域内，广泛分布着典型的别墅，但其大门均为上锁状态，难以进入，尚不确定其中是否存在物资或实体。

"雾中精灵，升腾飘影。"
[[/div]]


++ 绿化林

   截止2025年4月30日以前，绿化林普遍被认为是大厅的一部分，但经过多次勘探与大量流浪者的牺牲和贡献，**绿化林**最终被认为是一个单独的区域。

   当流浪者长时间探索**大厅**区域时，将不可避免的进入**绿化林**区域，而这个过程大概需要数分钟至数个小时。

   在外观上，该区域类似于前厅中的森林，主要由松树等常绿针叶植物组成，其入口附近的能见度约为5m。该区域的温度恒定在-12℃左右。据报告，身处其中将感受到轻微的不安感。

   值得注意的是，该区域的信号强度将随流浪者的深入逐渐减弱，最终断开；进入其中的流浪者报告到的能见度似乎也符合这一规律，而失去信号往往意味着该名流浪者的失踪。

   进入该区域的流浪者的报告指出，当处于**绿化林**区域中时，方向感将被弱化，使得流浪者极易迷路，尽管不排除能见度过低所致，但目前主流观点认为方向感弱化与层级效应息息相关。

    目前仅有一例进入**绿化林**区域后成功回到**大厅**区域的案例，此外还有大量流浪者在进入该区域后一段时间失去联系，并彻底失踪。

   鉴于目前对该区域的探索极为有限，且危险源尚未探明，不建议流浪者进入该区域。

+ 行动准则

++++ 应当

**·尽快切出该层级**
**·尽可能保持体温**
**·实时检查体温与环境温度**

++++ 不应

**·在原地长时间停留**
**·对该层级过多探索**
**·进入绿化林区域**

-----

+ 基地、社区和前哨

由于层级特殊效应，本层级不存在已知的基地、前哨或社区。

-----

+ 入口和出口

++ 入口
   1.在任意层级中回忆雪中的场景都有极小概率陷入昏迷，你将会在该层级的**回望步道**醒来。
   2.当你正在探索[[[Level IF-2]]]或[[[Level IF-18]]]时，可能将遇到一片能见度显著降低的区域，如果你试图对该区域进行探索，最终将会来到该层级的**回望步道**区域。

++ 出口
   1.据逃脱本层级的流浪者描述，在**大厅**区域切入一片**特殊的**雪花将会来到IF层群中随机一个降雪的层级，但目前无一人可以描述该片雪花的特殊性究竟如何。
   2.有传言称，若你在该层级停留至降雪结束时，将会来到该层级的某个子层级中，但该说法未被证实。
更多出口正在探索中...
