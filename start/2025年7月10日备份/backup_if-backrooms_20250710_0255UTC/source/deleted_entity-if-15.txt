**编号：IF-15**


+ 描述：
Entity IF-15，一般被称作“赛璐珞”，往往会对流浪者表现出较大的敌意，该类实体的行为为流浪者的生存构成了严重的威胁。目前，绝大部分个体都分布在外观类似于工厂的层级中。由于该实体的高度危险性与自我复制属性，M.E.G.正在积极研究该种实体的反制对策。多数直达该实体所广布的层级的通道已被封锁。

+ 生物学特征：
"赛璐珞"是一种大型人形实体。目前观测到绝大部分个体身高约为2.2~2.3米，极少见有超过2.8米者。其外观类似于一个人体模型，但往往会具有一些人类不可能存在的生理构造。比较常见的是，有的实体会具有可以触及地面的细长手臂，亦或者是有着长而弯曲的脖子。"赛璐珞"没有五官，只会在人类的眼睛处出现一个可以用肉眼明显观察到的凹陷。

"赛璐珞"的皮肤表面是一种化学合成材料，从外观上来看是透明的，具有磨砂质感，放置在灯光下时会呈现出哑光光泽。它具有许多独特的性质。通常比塑料更薄、更轻。当加热到较高的温度时，它会变脆并可能开裂。虽然比塑料更脆，但它仍然可以弯曲或扭曲。一些皮肤碎片具有较高的易燃性，因此在某些层级中较热的区域里常常能够看见出现自燃现象的"赛璐珞"。在高温下，个体可能会释放出一些酸性物质，产生像盐酸的味道，有些时候也可能产生类似于萘的味道。"赛璐珞"的皮肤表面柔韧性极好且富有弹性，但湿气、极端温度或某些化学物质都可能对其造成破坏。长时间储存在密闭环境中也可能导致其皮肤变得更脆，甚至开裂。

"赛璐珞"体内没有任何器官，由一种黑色㬵状物质填充而成，在运动时会产生体积与密度的变化，类似于生物肌肉组织的收缩与舒张。对这种物质进行化学成分的分析结果显示，其中可以检测到的化学成分有乙醇及萘（即人工樟脑），对该物质进行蒸馏与干燥后会得到一种黑色非晶体块状物，与黏土十分相似。当个体流失了过多这种物质后将不会再表现出生命体征，通过燃烧破坏其生理结构对于攻击该实体尤为有效。支撑实体胶状物质的是一种类似于钢筋的支架结构，其构成物质为一类复杂的合金，这一类合金的主要构成成分是硬铝以及一些未知成分，其表现出来远超一般合金的抗腐蚀性和柔韧性。

尽管该类实体没有正常的生理结构，但其仍然拥有良好的视力以及听力，并且智力接近于成年人。该方面的一个重要体现就是，它们对流浪者进行攻击前集体将其包围。出于这个原因，"赛璐珞"往往会对尝试逃跑的流浪者构成严重威胁。

+ 行为：
"赛璐珞"似乎有着群居习性，在某些地带中可以发现"赛璐珞"的大量聚集，但是独处的个体亦不罕见。通常情况下，落单的个体喜欢在走廊和房间中无意义地游荡，并且尝试使用钝器（通常是自己的手臂下段的部分）殴打经过的流浪者，再利用自己的身体组织将该名流浪者的尸体溶解得到一种夹杂着某种人造纤维的黑色的油状物质。流浪者体内不存在这种物质，这种纤维的来源为"赛璐珞"攻击流浪者时产生的，可以使生成的黑色油状液体增稠，对这种人造纤维进行化学成分分析类似于芳纶。之后，再由这些油状物质重新组成一个新的个体，这一过程通常在20~30分钟内完成，往往被认为是进行自我复制。

该类实体值得注意的另一类行为是，它们貌似喜欢停留在气温较高的地带，尽管这可能会导致它们的自燃或者表皮损伤。当温度升高时实体的活跃性会显著增长，这一类现象的成因还在研究中。

通过长期的观察，该类实体的敌意似乎正在逐渐增长。由于流浪者的死亡能够加速"赛璐珞"的自我复制速度，M.E.G提醒您：远离一切外观类似于人体模型的物体，尤其是当其出现不合理的结构时。

+ "穿行于工厂的孤独旅人请留意身后，
+ 不要掉入他们塑料做成的血盆大口。
+ 他们对受害者的窥视永远不会停息，
+ 即使他们光滑的面孔背后没有眼睛。"

+ 发现记录：
该实体首次于2018年7月13日被发现于某层级中，是较早的IF层群原生实体。目前有关于它的推测是，"赛璐珞"异常的化学构成与其不可思议的生物结构是由一次化工事故后自发产生的。

+ 行为准则：
**应当：**
警惕拥有异常身体比例的人影。
远离气温较高的地带。
竭力避免落单。
被威胁时尽力反抗。

**不应：**
放弃希望。
主动挑战它们。
在较热的区域中逗留。
徘徊在有阴影的走廊中。

+ 附录A：Entity IF-15-1事件报告
目前认为该实体已经产生了社会行为而并非单纯的聚集，因此该实体的危险性在一次观测事件后被认为正在上升。在这一次观测事件中，"赛璐珞"似乎采取了某种战术策略，对于Level[己编辑]中的某个流浪者社区进行攻击并且将其中的居民全部转化。
这个事件的具体过程如下：
一个"赛璐珞"跟踪了一名返回社区的流浪者，在这个过程中流浪者虽然有所察觉，但是同时也并没有在意。夜间社区集体休息期间，这名流浪者首先被同化并且引起了连锁效应，周围的实体群体包围了这个社区。事件最终导致这个社区内132名人员全部身亡。"赛璐珞"的实际危险性还有待评估。

+ 附录B：一名流浪者的日志
这一本日志被发现在事件Entity IF-15-1的发生地点，其内容披露了有关于该事件的一些细节。普遍认为这一本日志的书写者是上文中所提及的被某个个体跟踪的流浪者。以下为日志的内容。

“今天晚上回来的时候，发生了一些不太寻常的事情。

哈，没错，这就是在Level[已编辑]中。我们当然知道这里不太安全，如果不是生活所迫，谁愿意留在这里？

今天我如同以往一样穿过那一条充满了温热蒸汽的走道，其实每一次穿过那条走廊就会使我不那么高兴。我已经在这里生活了很久了，但是这里没有归属感可言。充斥着铁锈味的空气，还有那永远不会停息的机器运作的声音，时时刻刻都在警告着你，告诉你似乎失去了一些与生俱来的权利。

当时我正在滚烫的石砖地面上行走，这一片区域很热，它给我一种不祥的预感。我绕过了几个弯，听到后面有一些奇怪的声音。但是当我抬起头时发现，那是有水在我头顶上的金属管道里流动。虽然这一些声音听起来不怎么自然，但是我也不想在意了。

我一直在想，作为一个工厂城市的居民（虽然我很快就不是了），我实在是没有办法真心赞美这里。其实单从这里从未消散的臭水沟味，恶劣的生活环境就已经可见一斑了。我的脑海中常常浮现出一幅画面，就如同风景画一样宏大，只不过是关于一个主题——沸腾。

我能说些什么呢？随处可见的热水池，富磷的水体，水上堆积的像是草坪一样厚的工业废料在荧光灯的照耀下反射着令人作呕的色彩，堆满灰尘的工作室中到处都是垃圾。而我周围的某个邻居，他的孩子们，以为这才是真正的海洋世界哩。他们在水里玩闹，嬉戏，甚至潜到水里去探宝。那些漂浮在沸水上的塑料残片，像极了童年时母亲禁止我触碰的毒水母（现在想来那个叫氰水母吧）。如今它们已在这里繁殖成一片乳白色的群岛——孩子们踩着泡沫板在其中跳跃，把聚乙烯的触须缠在手腕上当作首饰。这种生活当然是现代人所难以理解的，只不过对于他们这一代出生于此，成长于此的人来说并不是什么大不了的事情。

我们也不是没有做过努力。我的另一个邻居，很了不起，围起了一片热水池，把上面的黑色残渣全部清理掉，让水露出它本来的面貌。他曾经得意洋洋的向我夸耀，‘这里也不过如此嘛。’只不过好景不长，相对洁净的状态只维持了三天。最终他放弃了，离开了这里，几天之后我在沸腾的水池中发现了漂浮在水面上的几缕彩虹色的头发，摸起来像是熔化的信用卡。我猜这就是那个染过头发的邻居留下来的遗物。我很确信的是，我再也没有见过他。

平静下来的时候，我会问自己到底发生什么。炎热的区域成为了滋生诡异生物的温床，这一点最好的体现就是，当可怜的孩子们度过了一段快乐的‘海滨时光’后，一些黑色的粘液爬到他们的身上。他们的皮肤红肿发炎，就像是水沸腾一样。他们急忙用这里的水去清洗，可是情况却越来越坏了。我们在水里投各种各样的化学品，希望能将水池里的罪恶清洗干净，可是都无济于事。多么令人绝望——不是想到自己的未来遥遥无期，而是明明做出了努力，也不能改变这里的情况，哪怕只是一点点呢。

我有一个自己的小屋，可以在里面过自己想过的生活。但是生活在这种环境里又有什么意义？既不舒适，亦不自由，最可怕的是难以离开。我可能会在哪一天死掉，连自己的尸体在哪里别人都不知道。周围有各种怪异生命在潜伏着，他们长得很像人，他们根本就不是人。他们简直是一团由塑料构成的，难以理解的生物。他们连脸都没有，他们总是想杀了我们。他们潜伏在水池下，视线穿越漂浮在水面上的垃圾监视着我们。

这里已经不能久待了。水面上漂浮着活体，水下还有不知道多少虫子在蠕动着。再看到那些戏水的孩子们只让我感到反胃。他们不懂得怎么保护自己，也不懂得真正的生活到底是什么样。这里的人们总有一天会因为各种各样的原因死亡，不过那和我已经没有关系了。我决定就在明天，我要带上我的东西离开这里，和这里彻底划清界限。

你问我为什么愿意放弃这里的家？这才不是家，这只是一个房子。这里就像是温水煮青蛙一样，把我们困在安全的假象中，一直到锅里的水沸腾，我们所有人都会被煮烂，皮肤被水泡得发胀。而从我五年前从地上掉入这些鬼地方开始，我已经把家放弃一次了。现在我为了更加光明的生活，而放弃这片烂泥塘，又有什么关系呢？

回来的时候，我再一次听到了奇怪的声音。当我回头的时候，只发现了荧光灯照射出的光线在墙壁拐角的地方画出一条光和黑暗交界的线。可能还有一个人影，但是当时已经不允许我再记住这些细节了。我得离开，我必须要走。先是从美国人海厄特发明早期塑料的制造技术开始，罪恶的塑料就开始统治这个世界了。多么可悲，即使在这里也是一样。

只不过，我隐隐有一种感觉，今晚可能会发生什么可怕的事情。
