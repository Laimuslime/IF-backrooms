+ **描述**
此层级的外观常被描述为一座永远处于深夜的城市[[footnote]]一般称本区域为主城区[[/footnote]]，城市内完全看不到任何一丝有关自然的景物，均为人类们的杰作。城市内的一切设施完善，但并不会运作，一切尝试恢复运作的办法均以失败告终。流浪者会在本层最大的十字路口醒来，街道两旁的路灯不会发出任何的光芒，两侧的商铺的招牌绝大部分已停止运作，不再发出任何光芒，显得整座城市死气沉沉。绝大多数的建筑物通往虚空，而极少量的建筑物可以进入，其内部通常表现出了被大火焚烧过的痕迹，并散发出阵阵煤油味。经过测量，凡是可进入的建筑物内部的一氧化碳含量均大幅度超过了人体最大极限值，因此不建议流浪者向其内部深入探索。

层级内唯一的光源为天空中的圆月与极少量正常运行的招牌，加上本层有着极为浓厚的雾霾，使得任何光源难以穿过雾霾，这也难以对此层级进行探索。当流浪者从任意方向行走，会发觉周围的景物在不断重复叠加，最终回到初始点，建筑物的轮廓并不会因环境或距离而变化，永远保持着模糊状。

据成功离开本层的幸存者之口述，推测本层的雾气极有可能会损害人体，所有幸存者的肺部均发现了不同程度的损伤。此外，本层氧气含量相近于地球氧气含量。但由于本层之雾霾，这意味着流浪者误入此层后应当速速离去。

此层级不存在任何昼夜循环，温度永远恒定在1°C左右。此层级的氧气含量正在以未知原因下降，但实体数量却依然在增多。此外，此层级存在孤立效应，这对于流浪者是一个巨大的挑战。

除环境之外，实体是本层最大的威胁。这些实体往往看不清其五官[[footnote]]故称其为黑影[[/footnote]]，仅有那一团似人的黑影。黑影完全无视环境带来的危害，依旧可以正常攻击流浪者，流浪者此时要做到的是保持冷静，积极寻找出口，不要放弃。

当流浪者行走或逃跑过程中，若周围景色不再重复交替，黑影数量急剧减少，则会到达以下两处区域中的某一处。这些区域通常被认为是本层的隐秘区域，其内部之场景与本层主体区域有着极大的差异，曾一度被认定为是未被踏足过的层级。但由于两处区域存在种种不真实性，类似于梦境中的想象之地，故归属于本层。

++ **森林**
[[include component:image-block
 |name= http://brsandbox-mini.wikidot.com/local--files/57-57/%E6%A3%AE%E6%9E%97.jpg
 |caption=**森林**
]]
**森林**，通常被描述为一处永远处于深夜的森林。其内树木交错、杂草丛生。此区域同样保留了主区域之阴森，其中的水气含量以稳定的速率上升，导致暴雨不断，淹没整片森林。但若流浪者重新进入本区域，一切将恢复原状[[footnote]]而这并不会对主区域产生影响[[/footnote]]。因此，流浪者务必尽快找到出口。

此层级虽植物繁盛，但氧气含量稀薄，二氧化碳的浓度恒定不变，且永远大于氧气含量。据流浪者在本层的探索，发觉本区域内的一切植物均为空壳，植物样本上没有发现任何细胞的踪迹，如同人造植物，乱中有序地摆在森林之中。有学者推断，在本区域存在一类未知生物在呼吸这里的氧气，目前关于未知生物是否存在[[footnote]]一部分人相信数据，认为测量准确，相信生物之存在。而另一部分人认为，如此稀薄的氧气根本无法供未知生物存活，因此不断争论[[/footnote]]仍在争议中。小部分学者认为本区域本身就是一类巨大的实体，但没证据证明此观点。
+++ **如何进入**
* 若流浪者于本层任意一条道路行走时，发觉周围环景忽然间变为一片黑暗，一段时间后将来至此区域
+++ **如何离开**
* 在本区域行走时，若周围环境再度变为一片黑暗，将返回**主城区**
* 当满月降临时，可前往**坟场**

++ **坟场**
[[=]]
**传说，当森林和主城区的月亮显出其全貌，即可来到此处。**
[[/=]]
@@@@
[[include component:image-block
 |name= http://brsandbox-mini.wikidot.com/local--files/57-57/%E5%9D%9F%E5%9C%BA.webp
 |caption=**坟场**
]]
一栋硕大的诡异房屋便是此区域的标志，通常认为，诡异建筑是本层的中心，也是所有流浪者进入此区域后首先来到的必经之地。中心处周围散落着几颗枯树，为此区域的诡异添了几笔点睛。枯树周围散落着零零散散的无字碑，而其地下则是无尽的黑暗虚空。枯黄的野草便是那脆弱的屏障。那明亮的圆月，枯燥乏味的天空，永不消散的薄雾，便是此区域最标志性的景色。

在进入此层级后，所有人造设备均会出现不同程度的损坏，这导致无法测量本层的详细数据。此层级的氧气含量足以让人感到舒畅，然而流浪者在初入此层级会莫名抑郁，且无法缓解，最后所有流浪者都跪在一座最大的无字碑前，绝望死去。因此，此区域也被认定为是此层级最危险的区域，也是最为神秘之区域。

当流浪者行走足够的距离[[footnote]]无明确距离[[/footnote]]，可以来到本区域之边界。边界由一道高大的砖墙与外界间隔。其类似于古代城墙，残破的石砖上布满了岁月的苍痕。砖墙疑似为无限延伸，尝试翻越砖墙的行为均已失败告终。
+++ **如何进入**
* 当**森林**与**主城区**的满月现身，即可来到本区域
+++ **如何离开**
* 穿过本区域的迷雾区可返回**主城区**
* 传说，从木门穿过砖墙即可通往黑暗，不过并未被证实

+ **实体**
++ **黑影**
* 具有人类外观
* 身体通黑，不见五官
* 对人类有极强的攻击性
+++ 应当
* 若附近存在黑影[[footnote]]黑影具有特殊性质，若流浪者察觉附近存在黑影，则流浪者会感觉恐慌，被人注视之感[[/footnote]]，弯腰静步，缓慢离开
+ **基地，前哨和社区**
此层级无任何已知基地，前哨和社区。
+ **入口与出口**
++ 入口
* [https://backrooms-wiki-cn.wikidot.com/level-11| Level 11]中被大火焚毁的房子进行切出即可来到本层
* 从[[[Level IF-2]]]中造型诡异的树木切出可以来至本层
++ 出口
* 从水管切出可前往 [https://backrooms-wiki-cn.wikidot.com/level-c-593| Level C-593]
* 从本层的别墅切出可返回[https://backrooms-wiki-cn.wikidot.com/level-11| Level 11]
* 从本层的建筑废墟切出可前往[https://backrooms-wiki-cn.wikidot.com/level-11.3| Level 11.3]
* 从本层有光源的建筑[[footnote]]此类建筑占本层建筑中的较少部分[[/footnote]]切出，可前往任意安全层级
