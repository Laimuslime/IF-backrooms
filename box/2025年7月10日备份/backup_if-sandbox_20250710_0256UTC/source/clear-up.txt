= 醒了。

昨晚下了一整夜的雨，早上窗户外已经放晴，天空蓝得刺眼。镜手最后拉上窗帘，房间里一片昏暗。

她打开手机，被光照得眯了眯眼，调暗亮度后发现消息栏上挂了一整晚的工作告示。

= “又搞什么东西。”

说着，镜手点开短信，上面是：

= “鉴于目前M.E.G. 勘探人手短缺，由于表现良好，特调动外交部工作人员Mirror206 执行任务#206-E 。具体内容请前往Level AS-1 速切中心查看。收到请回1。”

= “TD。”

镜手没能想到自己工作的岗位有天也能被傻逼高层调去卖命，而且不涨薪水。她烦躁地叹了口气，安慰自己至少没给自己安排到其他层级去。速切中心就在她家对面，过一个红绿灯就能到。

= “红灯，请留步。”

然而这一块的红绿灯和她的高层一样傻逼；红灯拦人六十秒，绿灯只有十秒，她只能跑着冲过斑马线。幸好这里鲜少有不守交规的车辆，否则镜手怎的也得被撞个人仰马翻。

= “绿灯，请通过马路。”

任务内容相对容易，是坐速切玩家的船去群岛勘查新发现的岛的情况。蓝海通常不能通过切行穿越，所以即使性急如速切也得慢慢坐船漂过去——但愿自己不会晕船。

----

镜手在港口百无聊赖地看着一艘艘船从远方驶来，一艘艘船从这里驶去，上面下来奇形怪状的生命。“船”其实不是船，内部架构和用来航普通海的船已经风牛马不相及，只有形状同船相似。我们下面还是称“船”叫“船”。虽然没有海的各种影响因子，不管感觉上还是实际上，船状的船要比其他形状的船快，木头的船要比其他材质的船稳，所以依然可以在港口看见复古的帆船——船加上帆也会更快——和摆渡的小船，甚至有木筏和羊皮筏。

远处就是蓝海，比她讨厌的晴天的蓝幽深许多，导致镜手更急了。从港口出发到进入蓝海，还有一段虚空，可以看见船只用各种方式来使自己浮空。

= “你好。”

椅子上，镜手的肩膀被拍了一下。她回头一看，是於复生，勘探部的人。

於复生身材高挑，穿着白短袖和牛仔裤，头上的白色遮阳帽用丝带打了一个蝴蝶结，侧面插着黄蕊小花，色调和镜手的一身黑灰形成反差。

镜手听过她。

= “嗯……你和我坐一趟船？”

= “我和你做一个任务。”

镜手和她都没再说话。於复生坐在椅子上对着黑屏的手机发呆，旁边那位不耐烦地看表。

= “延误了？”

镜手正这么想着，耳边传来响亮的号子声。船来了。

那是一叶摆渡的小舟，上路的人只有镜手和於复生两个。

----

蓝海凉飕飕的。进入深蓝的时候，镜手感觉全身一阵放松。晴天太扎眼了，由于自己工作并不繁忙，镜手在这种天气里坚决不拉开窗帘或出门，而是窝在十六度空调的家的被子里玩笔记本。她现在心情很不好。

= “你很讨厌晴天吗？”

於复生凑近，轻声来问。

= “啊……是——诶诶诶？”

小船蓦地一阵颠簸，身后港湾的闪烁肉眼可见。镜手脑袋有些发懵。

= “喂？你还好吗。”

= “咳咳……还行。”

於复生看了一眼船夫胸口速切的标识。

= “去群岛的船都会这样。这是种切行技术，掌握的数速切玩家最多。不然路太远会很难办。当然，很伤身，你回去可以看看组织给你的报销单。”

镜手点了点头，眼前仿佛是一盘正在胡乱搅和的颜料，引得她晕眩了好一会儿。

= “下次别去群岛了。很不幸地，你返程的时候还要再坐一次这种船。”

= “？！”

----

镜手收起了手机。周围起初还能看到被蓝海淹没的烂尾楼，随着远去，抬头四望便只能看见深邃的幽蓝，仿佛自己真正飘摇在无波的海上，信号也愈加减弱，船夫依然摇着橹。这样，她感到一阵莫名的空虚。

= “我们现在正在这层海平面的正中央。”

於复生冷不丁地开口，她被吓了一跳，紧接着目光注意到她手上的屏幕。

= “这什么？”

= “勘探部特地给我们这些人配备的定位器，大部分情况下都运行正常。——非常贵，也很有用。”

於复生见她不是很乐意听，也识相地闭了嘴。镜手正琢磨有没有方法能让自己躺着睡一会儿，但若如此便只能将头枕在於复生腿上，还是算了。

小船配有安全带，因此不用担心没到站就被迫去海里游泳。镜手安心地斜着头，进入梦乡。

----

镜手一觉睡到站，睁眼时已经可以望见前方云一样的岛屿。她确实错怪同事了，除了刚开始巨大的颠簸外，全程都没有遇到什么意外，下了船也只有轻微的不适。岛上正在下雨，柔和凉爽的雨丝让镜手感到舒适，心也凉快起来。

於复生在一旁摆弄各种器械。镜手不知道她是怎么把这么多东西带在身上的，甚至压凹了岛屿棉花糖一样的地面，或者云。

= “我感觉这地方明天就要散了。”

= “确实。”

於复生盯着眼花缭乱的字符串作出肯定。

镜手其一逃避坐船；其二不想回去在晴天底下待着；其三是知道蓝海淹不死人；於复生则想休息一会儿。两人都想多在这里多待点儿时间，毕竟这样清新的空气可不多见。

言归正传——这个岛屿总的来说是一大团云，其中有乌云和其他各种云，还生有一些攻击性并不强的实体。譬如说云妖，会吞雾吐云，发射威力不大的云弹；粘在一起的两团放电乌云，碰到的流浪者会浑身静电；等等。

你可以看见水珠从乌云里飞上天，然后作为雨降到各处。

镜手刨了一下这些云，凉飕飕的，脱离整体后变成雾状飞上天，下面露出有些粗粝的沙子，类似[https://backrooms-wiki-cn.wikidot.com/level-48 沙滩]上的那种质感。

= “对的，我刚刚测了，这是一个复合岛屿。如果不错，明天我们就只能看见云底下的沙滩以及岛屿中心的草地——如果那些[http://backrooms-wiki-cn.wikidot.com/ 讨厌的石头]不来捣乱。嗯，散开的云可能飞到天上直接导致一场大雨，我们得做好迎接狂风暴雨的准备——帐篷应该完全可以撑住。”

镜手没有接话，视线随着一望无际的白与灰远去。

----

= 帐篷里。

= “好黑。”

= “我们来得不巧，太阳现在还在岛屿底下呢。差不多还有六个小时，天就亮了，到时候可以出去。以我的经验，光底下远比黑洞洞的地方来得安全。”

镜手点起煤油灯，让帐篷里有了亮光。其实岛屿并没有全黑；对于这种体量不大的岛，它们自己的太阳的光可以射向四周，使天边生出几环颜色。这正是神奇的地方：岛屿多数没有大气层之类，但霞光就在那，而且各自有各自的颜色。像这座岛便是金色。所以，一些群岛住民会将午夜太阳被压在底部时所呈现的霞环作为边界，以区别各岛岛民们的领土。

镜手并不讨厌早晚的晴天，就像这时，因为那种光照很温和，像是温暖给予世界的温柔的怀抱。

= “所以我们就在这里一直等到这团云被吹飞吗？”

= “也许呢，守则上说如果勘探目的地损毁可以提前返回，但反正你也不想回去。”

信号不好，卡顿。镜手无聊地睡了过去。

岛上蓦然刮起一阵狂风，整个帐篷都剧烈晃动起来。於复生放下手机。镜手被吵醒了。

= “看来我们要提早准备返程了。我手机完全打不开，你应该可以约到船吧，复生？”

於复生说不急，速切的船不管什么时候打都来得及。

风从东面来，云被吹到西边水墨画一样由乌云组成的高山，并且还在变得愈发高大雄伟。

= “我感觉一会儿要下雨。”

= “你直觉很准。还有几个小时风向会反过来，现在那边的云山就在下雨，到时间会吹到我们这儿来，太阳同时升起。”

= “出去逛逛么？”

镜手盯着外面狂暴的大风和向西山飞去的云雾，点了点头。

----

她们飞起来了，真的。按理说这种级别的风吹不跑人，但是镜手和於复生的体重变得和那些云相比不重多少。这样，她们理所当然地与风云同去。

身下一片乌云的灰黑，身后矗立着一座高山，身前是黎明的淡淡金粉。两人起初还有些惊慌，之后想到自己最终要落在弹软的云山上，最次也是飘在蓝海的虚无里就安静下来，脸朝风而身子向后飘去。风从脸旁簌簌地刮过，让镜手忍不住打了个寒颤。

地面被卷出底部的草原和沙地。云雾弥漫。

= “我就不该跟你一块出来。冷诶。”

= “喝点杏仁水就好了，等落地了再拿，小心飞掉。”

她们发出的声音异常清晰。

----

= “哈……呼……冷死了！”

两人终于落到山上，头发张牙舞爪地向后逃去。趴了一会儿，她们坐起来，手撑着地去看下方狂乱的景象。

= “……所以，有任务报告这回事吗？”

= “没有的。原则上复合岛屿分解后属于新岛屿，我们的任务已经结束了。——况且这里没有什么探索的必要。”

日出东方，风的位置让给朝霞和金光，它从西面吹来，正在两人身后。

= “哦……风向怎么变了？这些风到底是从哪来的？”

= “不清楚，但咱一会儿又得飞起来了，我的身体正在变轻。”

这时天上已经下起淅淅沥沥的雨，斜着打去。

在帐篷里休息一阵，於复生滴了一艘船，船夫还是来时那个——镜手直到现在都没记住船夫的样子。

一觉之后。

= “呼……总算回来了啊。”

镜手看着不远处蓝海的边界，船夫正预备着打开飞行器进港。几十秒后，岸上出现两人的身影。镜手伸着懒腰，困乏地说：

= “啊，所以你为什么说那地方没必要看啊？”

= “因为这样啊。”

於复生伸手将镜手拥进怀中，然后推她坠入虚空之下。

----

= “啊呀呀呀——於复生！”

镜手猛地从冰冰凉凉的台子上坐起，头磕到上面莫名的器具，接着在疼痛中听到自己几位朋友的交谈声。

= “哎呀，镜手醒了！”

她起初有些发懵，后来想起自己上午被组织叫去测试心理健康，最后喜得排在下午的梦境心理调节技术试验一次——而且费用全报销。

事实证明疗程确实有效，镜手感到自己心里已经有些放晴。后来黄昏了，朋友都散开，却又感到有些难过，像被什么抛弃了一样。

----

最后从医生口中，镜手得知：这一块梦境是一个真实存在的於复生自己演的，她本人身份和形象也的确和梦中相差不大。之后，她又听说：主管，也就是Cattt，刚好有於复生的联系方式！

经过一系列死皮赖脸的讨要，镜手最终拿到了於复生的速切论坛账号“Anrifels_sunday”——是的，她同时还是速切玩家。

= “你好？”

= “Hi.”

= 意料之外的秒回。

= 这才是两人的第一次对话。
