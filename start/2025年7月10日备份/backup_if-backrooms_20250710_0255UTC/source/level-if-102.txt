**Level IF-102**是后室IF层群中的第102个层级，由于未知原因，流浪者在切入该层级时必定身处层级的正中心。


----
[[=]]
+ **― 描述 ―**
[[/=]]
----


**Level IF-102**是一片面积约为3600平方千米的花海，绿草与鲜花交织在一起，构成了该层级的主要地貌。经调查，该层级的所有鲜花均属于菊科植物”[[footnote]]雏菊区的绿草除外[[/footnote]]，不同的花卉将该层级分割成了三个区域。所有区域都风和日丽，温度永远保持在21摄氏度，天空永远是湛蓝的，永远保持在白昼，且自该层级被发现起，没有记录到除晴天以外的任何一种天气在该层级出现过，但这并不影响鲜花的盛开。

**Level IF-102**的内部除了平原地形之外，河流和山脉在该层级也并不罕见，天空中偶尔会有几朵蒲公英或几只麻雀飞过[[footnote]]蒲公英因未知原因可以在无风情况下飞行，但只能持续3-7分钟[[/footnote]]，在落地后，它们会立刻消失。该层级的三个部分分别遍布着向日葵，雏菊，雪绒花三种花卉，这三个区域相交的点即为层级中心。该层级的花朵均可以被破坏，但在被破坏后会在4-8秒后原地重新长出，该层级的所有花卉均可食用，且味道鲜美，因此在该层级有着无尽的食物。

++ 向日葵区

该层级中由向日葵构成的区域名为向日葵区，面积约1100平方千米，是三个区域中最小的区域。该区域布满着向日葵，向日葵的高度一般在2.1米-2.6米之间，正因如此，该区域的部分地区会被遮挡视野，且由于向日葵的密集性，无法在向日葵区设立基地与前哨。向日葵永远向着太阳的方向，除非令其死亡，否则没有任何方法令其朝向太阳以外的任何方向。

该区域的向日葵在食用后可以令流浪者在3分钟内不受到“迷路效应”[[footnote]]雏菊区的一种特殊现象[[/footnote]]影响 。   切记！不要在向日葵区使用任何人造光源，这样做会导致流浪者被未知来源的高热射线照射。

++++ 向阳效应

向阳效应是向日葵区的一种特殊效应，具体表现为流浪者随着在向日葵区停留时间的增长，逐渐对太阳产生喜爱，并对月亮感到厌恶，但这种现象并不会令流浪者迷失自我，仅仅会令流浪者不想离开向日葵区。由于向日葵区的环境并无明显危险性，且该效应会在离开向日葵区的一瞬间消失，固此M.E.G.将其归类为了无害现象。

++++ 温暖阳光

温暖阳光是向日葵区的一种特殊且有利的效应，流浪着在受到阳光照射时会感受到身心温暖，一些程度较轻的恶性精神影响也能被减弱或消除，同时该现象还会大大加快流浪者伤口恢复的速度，据后期统计，约为正常速度的11倍。任何外来实体，包括友善实体，在接触到阳光的一瞬间便会灰飞烟灭，因此M.E.G.尽可能地让友善实体不进入向日葵区。

++ 雏菊区

该层级中由雏菊构成的区域名为雏菊区，面积约1200平方千米，是三个区域中中等大小的区域。该区域布满着大量的雏菊和各种绿草，两者在该层级的比例大约是3比4。该区域的绿草和其他花卉相同，具有再生性质，但含有未知毒素，不可食用。与绿草不同，该区域的雏菊与向日葵和雪绒花相比，营养价值是最高的。

相对于向日葵区和雪绒花区，该层级的蒲公英和麻雀更为常见，同时麻雀和蒲公英的飞行速度
更快，即使雏菊区的风力与其他两个区域相差无几。同时雏菊区更容易触发该层级最安全的出口[[[https://asbackroom.wikidot.com/level-as-99|level-AS-99]]]”[[footnote]]向日葵区的蒲公英常常落在高处，雪绒花区的蒲公英不常见[[/footnote]]
++++ 迷路效应

迷路效应是雏菊区的特有效应，流浪者在受到该效应的影响时会如同在非欧几里得空间一般严重的迷失方向，也就构成了我们常说的迷路，目前缓解该效应的唯一方法是食用向日葵区的向日葵至少50g。为了对抗该效应，M.E.G.在雏菊区设立了大量的指示标牌，若流浪者迷失方向，请遵循指示牌的指引，方可辨别方向。

++++ 命运投影

命运投影是雏菊区的一种极为有利的现象，其会在流浪者在受伤/精神受损/死亡/切出层级的6分钟前为流浪者投影出事件发生时大致内容，方式与[[[https://asbackroom.wikidot.com/phenomenon-as-4|phenomenon-as-4]]]相似，但其不会令流浪者强制遵从投影，流浪者可以通过该现象逃避危险，改变命运，M.E.G.推荐所有流浪者利用该现象逃避意外。

++ 雪绒花区

该层级中由雪绒花构成的区域名为雪绒花区，面积约1300平方千米，也是三个区域中最大的区域，与其他两个区域不同，该区域不存在明显的特殊现象，唯一较为异常的一点为蒲公英和麻雀在雪绒花区的目击报告大约是雏菊区的七分之一，由此可见蒲公英和麻雀在雪绒花区中较为罕见。

雪绒花区拥有着在该层级中最特别的自然景象，不会融化的积雪，天空中飘落着雪花[[footnote]]天空本身依然湛蓝[[/footnote]]，但其既不会降低温度，也不具备毒性，且在触碰到任意物体后会立刻消失，因此M.E.G.认为其不属于降雪天气。

雪绒花在雪绒花区的出现频率是完全不固定的，有些区域两个雪绒花之间相隔20米，有些地区雪绒花则遍地都是。枯木零散的遍布在该区域，但它们实际由某种未知物质构成，无法被破坏，触碰这些枯树会导致流浪者被麻痹，这种麻痹并不会伤害人体，仅会带来痛楚。

[[=]]
+ **― 实体 ―**
[[/=]]
----
该层级存在一个未知实体，自称演奏者，其似乎对流浪者表现了中立态度，目前我们对该实体了解的甚少，暂归类为中立实体，若在未来该实体展现出了敌意，我们会考虑将该层级的生存难度提高至1。

+++ M.E.G.      方向标前哨

* 由M.E.G.建设，位于雏菊区的中心
* 11人常驻，一般不进行轮换
* 负责指引流浪者规避迷路效应
* 友好且开放贸易

+++ 后室娱乐   雏菊口香糖开发基地

* 由后室娱乐建设，位于雏菊区的最西边
* 96-102人常驻，经常进行人员轮换
* 负责收集雏菊以制作他们最新款的“雏菊口香糖”
* 中立但开放贸易

+++ 冰点社区

* 位于雪绒花区西北方向
* 大约7000人
* 是该层级唯一的大型社区

+++ 入口
* 在[[[level-if-40]]]（死区）中经历任何现象将必定切入**Level IF-102**

* 流浪者在[[[level-if-42]]]（未定分级）中自杀会切入**Level IF-102**，通过这种方法切入的流浪者不会被木制尖刺刺穿

* [[[level-if-35]]]（死区）中的蝉若从办公桌切出[[[level-if-35]]]后会来到该层级，以麻雀的形态继续生存

* 在[[[level-if-86]]]（变化）或[[[level-if-101]]]（5）中长时间凝视太阳/月亮有概率切入该层级

* 将[[[level-if-46]]]（宜居）中的收音机及循环播放至第25次时，完整收听的流浪者会切入**Level IF-102**

+++ 出口

* 若掉落的蒲公英刚好落到流浪者的头上则流浪者会切入[[[https://asbackroom.wikidot.com/level-as-99|level-AS-99]]]（0）

* 麻雀有时会突然飞下来啄瞎流浪者的眼睛，此时流浪者会切入[[[https://backrooms-wiki-cn.wikidot.com/level-501|level 501]]]（死区）

* 演奏者会帮助流浪者切入[[[level-if-1]]]，但这仅限于流浪者至少在该层级停留100天的情况下。
