+ 描述

Object IF-11，又称解忧罐头，是后室IF层群中常见的食物，但是其一般只出现在带有货架或房屋的层级中。解忧罐头的外表类似于普通的水果罐头，罐头种类与装填物完全随机。解忧罐头的外部印有彩色的“原制造厂”四字，内部液体极为浑浊，可以依靠这些来分辨其与普通罐头。解忧罐头在开封时会散发出一种迷人的香味，对人体无害也无益。
+ 特性
Object IF-11不会变质，在开启前不会被损坏、摔碎或松动，但是流浪者仍然可以轻易的打开罐头盖。解忧罐头在打开后即可被食用，其内部并非普通的食物材料，而是一种特殊的压缩食材，无味，在正常情况下解忧罐头可作为基本的食物和饮水来源。解忧罐头的最大特性是其可以在一定程度上对使用者产生影响，以下为不同类型罐头的作用：
* 橘子罐头：无效果，但是似乎可以刺激流浪者的某些欲望[[footnote]]这种效果完全是随机的，概率较低。[[/footnote]]。
* 黄桃罐头：可以消除流浪者受到的所有负面精神影响，包括负面情绪、精神障碍等，持续1小时，长期食用无害。
* 梨子罐头：可以治愈流浪者受到的小伤，也可以起到止血的作用，将梨子罐头汁液直接倾倒在伤口上也可以起到类似的作用。不过其似乎不能增加流浪者的抗受伤性。
* 苹果罐头：食用时会逐渐恢复流浪者的体力，使流浪者感到精神充沛，同时也可以缓解部分疼痛。
+ 发现
解忧罐头最早被发现于[[[Level IF-19]]]的货架中，当时一名流浪者试图带走一个货架上全部的货物，意外发现了大量印有特殊字样的罐头，在食用后发现了其效果并上报M.E.G.，后为其编号为11号物品。随后在短时间内，IF层群大部分层级都发现了大量的解忧罐头，其中多数会与其他普通罐头一起刷新，据传，当流浪者陷入迷茫和精神问题时，他们将更容易发现解忧罐头。
+ 补充
除此之外，部分解忧罐头的底部可以发现一些小型标语，以下是部分被记录到的标语内容：
> 居然被你发现了？看来我藏的还不够深呢。

> 作为你路途上最忠实的伙伴。

> 你已经很累了，坐下来休息一会吧。

> 旨在解决您的一切烦心事，我们是原制造厂！加盟电话1xx......

> 人生就如罐头一般，虽然能从外表分辨一二，但当你彻底打开它之前，你永远不知道它的好坏。

> 当你陷入困境，堕入低谷，请不要放弃，因为你的身后还有这个——解忧罐头！
